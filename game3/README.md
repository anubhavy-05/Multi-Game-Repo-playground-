# Flappy Bird Game

A fun and enhanced Flappy Bird clone built with HTML5 Canvas and JavaScript, featuring power-ups, coins, particle effects, and a dynamic day/night cycle!

## How to Play

1. Open `index.html` in your web browser
2. Press **SPACE** or **Click** to make the bird flap and avoid the pipes
3. Collect **coins** for bonus points (+5 each)
4. Grab **shield power-ups** for temporary invincibility
5. Try to get the highest score possible!
6. Press **R** to restart the game

## Features

### Core Gameplay
- Simple and intuitive gameplay
- Score tracking with local storage for high scores
- Smooth animations
- Responsive design
- Clean and colorful UI

### New Enhanced Features
- 🪙 **Collectible Coins**: Gather golden coins for +5 bonus points each
- 🛡️ **Shield Power-Ups**: Temporary invincibility that protects you from pipes and obstacles
- ✨ **Particle Effects**: Beautiful visual feedback for flapping, collecting items, and power-up activation
- 🌙 **Day/Night Cycle**: Dynamic background that changes from day to night every 50 points, complete with twinkling stars
- 📊 **Enhanced Stats**: Track your score, high score, and total coins collected

## Game Mechanics

- The bird falls due to gravity
- Press space or click to make the bird jump
- Avoid hitting the pipes or the ground
- Score increases as you pass through pipes
- Collect coins for bonus points
- Shield power-ups appear periodically and provide 5 seconds of protection
- Background transitions to night mode at higher scores for added visual variety
- Game gets progressively challenging

## Visual Effects

- **Flap Particles**: Blue particles appear when the bird flaps
- **Coin Particles**: Golden explosion when collecting coins
- **Shield Particles**: Blue shield aura and particle effects when power-up is active
- **Shield Timer**: On-screen countdown showing remaining shield duration
- **Night Sky**: Twinkling stars during night mode

## Technologies Used

- HTML5 Canvas
- CSS3
- Vanilla JavaScript
- LocalStorage for score persistence
